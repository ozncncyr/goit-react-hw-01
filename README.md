## goit-react-hw-01

Bu proje, React ile geliştirilmiş temel bir kullanıcı arayüzü uygulamasıdır.
Vite ile başlatılmıştır ve modern React fonksiyonelliklerini kullanır.

## Özellikler

- **Profil Bileşeni:** Kullanıcı bilgilerini gösterir.
- **Arkadaş Listesi:** JSON dosyasından alınan arkadaşları listeler.
- **Arkadaş Listesi Öğesi:** Her bir arkadaşın durumunu ve bilgilerini gösterir.
- **İşlem Geçmişi:** Kullanıcıya ait işlemleri tablo şeklinde sunar.

## Klasör Yapısı

```
src/

```

## Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/ozncncyr/goit-react-hw-01.git
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Uygulamayı başlatın:
   ```bash
   npm run dev
   ```
